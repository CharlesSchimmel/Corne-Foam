# Corne-Foam
SVG files for Corne V3 that include space for diodes. Based on ObliviousGmn's 
work

I made these in mind for IMK, But should work on other corne cases. 

Recommended Heights. 

Plate Foam: 3mm 

Back Foam: 2mm
